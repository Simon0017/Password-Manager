# PASSWORD MANAGER 

## About
This project is a C++ implementation of a password manager. Designed so that users can store and manage their passwords of different apps and sites. Offers encryption and hashing to protect the passwords. Comes with a graphical user interface designed so that the user can interact freely with the backend.

---

## Features
- Password hashing
- Data encryption and decryption
- Database storage (Sqlite)
- UI - *Generated by claude*
    - Login Page
    - Registration Page
    - Main Dashboard Page
    - Add password Page
    - Edit password Page
    - User Management Page
    - Message Box

## External Libraries
1. SQliteCPP
2. Dear Imgui
3. Libsodium
4. Cryptopp


## System Requirements
- CMAKe ^17
- Vcpkg manager
- C++ compiler

## How to run the project
1. ```bash
    git clone ....
    ```
2. ```bash
    cd "PassWord Manager CPP"
    ```
3. ```bash
    cmake -S . -B build -DCMAKE_TOOLCHAIN_FILE=C:/vcpkg/scripts/buildsystems/vcpkg.cmake
    ```
4. ```bash
    cmake --build build
    ```
